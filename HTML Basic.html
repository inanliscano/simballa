HTML Documents


All HTML documents must start with a document type declaration: <!DOCTYPE html>

All HTML document itself begins with <html> and ends with </html>

The visible part of HTML document is between <body> and </body>

The <!DOCTYPE> Declaration

The <!DOCTYPE> Declaration



The <!DOCTYPE> represent the document type, and helps browser to display web pages correctly.

It must only appear once, at the top of the page (before any HTML tags).

The <!DOCTYPE> declaration is not case sensitive.

The <!DOCTYPE> declaration for HTML5 is:

<!DOCTYPE html>

The HTML Links



HTML links are defined with the <a> tag, together with href attribute wich specify the link's destination.

Example:

HTML Image

The HTML Image



HTML Images are defined with the <img> tag.

Together with src attribute which is the source file, alt attribute which is the alternative text which will show up if the internet is slow and the image is having a hard time loading. In addition is the width and height attributes that allow the image's size to be manipulated.

Example:

<img src="image.jpg"alt="an image" width="200px" height="200px">
